# Aethrion v0.1.0-alpha

## What Is Aethrion?

Aethrion is a persistent social simulation runtime for AI characters that remember, relate, and act over time.

The core idea:

> LLMs generate expression; deterministic rules drive the simulation.

Aethrion models memory, emotion, relationships, and proactive behavior as a deterministic event-driven layer. LLM adapters are expression layers, not authorities over simulation state or rule outcomes.

## Included

- Public runtime API through `Aethrion.Runtime.dispatch/2`
- Structured runtime errors through `%Aethrion.Error{}`
- Scripted CLI demo through `mix demo.drama`
- Interactive CLI demo through `mix demo.interactive`
- Branched scenario demo through `mix demo.branches`
- JSON file persistence for local experiments
- Supervised runtime server through `Aethrion.RuntimeServer`
- Scheduler process through `Aethrion.Scheduler`
- Fake LLM adapter for deterministic tests and demos
- Bilingual README in English and Korean
- API, concept, MVP, and roadmap docs
- ExUnit test coverage
- GitHub Actions CI

## Alpha Caveats

- API may change.
- Not production-ready.
- No real LLM provider yet.
- No Phoenix UI.
- No vector database integration.
- No distributed runtime or per-character actor runtime.
- Not published to Hex yet.

## Demo

The interactive demo is available as an MP4 at [`assets/demo/interactive-demo.mp4`](https://github.com/simulacre7/aethrion/blob/main/assets/demo/interactive-demo.mp4), with an animated SVG fallback at [`assets/demo/interactive-demo-readable.svg`](https://github.com/simulacre7/aethrion/blob/main/assets/demo/interactive-demo-readable.svg) and a plain-text transcript at [`assets/demo/interactive-demo.txt`](https://github.com/simulacre7/aethrion/blob/main/assets/demo/interactive-demo.txt).

It shows this flow:

```txt
user gives Mina a flower
Mina remembers it
Yuna observes it
Yuna jealousy/tension increases
time passes
Yuna proactively messages the user
```

The branch demo also compares a user who ignores Yuna with a user who apologizes. The apology branch creates a reconciliation memory, increases Yuna's trust toward the user, and prevents the jealous proactive message.
